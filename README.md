# 🌦️ Weather App

A modern and responsive Weather Application that provides real-time weather updates using API integration. Built with a focus on clean UI, performance, and user experience.

---

## 🚀 Features

- 🌍 Get real-time weather data for any city
- 🌡️ Temperature, humidity, wind speed, and conditions
- 🎯 Accurate API-based weather forecasting
- 📱 Responsive UI (mobile + desktop)
- ⚡ Fast and lightweight performance
